# XamarinBlobStorageApp

A a single-page Xamarin.Forms app that displays an image from Azure Blob Storage by communicating directly with an Azure Blob Storage Container using the [Azure Storage SDK for .NET](https://docs.microsoft.com/dotnet/api/overview/azure/storage/?WT.mc_id=mobile-0000-bramin).
